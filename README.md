# LostRoutes-迷失航线手机游戏，一款简单轻松的非过关类第三视角射击游戏
##游戏主角是一架第二次世界大战时期的老式轰炸机，在迷失航线后穿越宇宙、穿越时空，与敌人激战的同时躲避虚拟时空里的生物和小行星。
